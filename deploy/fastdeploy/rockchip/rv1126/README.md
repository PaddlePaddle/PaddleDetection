# PaddleDetection 检测模型在瑞芯微NPU上的部署方案-FastDeploy

## 1. 说明  
本示例基于RV1126来介绍如何使用FastDeploy部署PaddleDetection模型，支持如下芯片的部署：  
- Rockchip RV1109
- Rockchip RV1126
- Rockchip RK1808

模型的量化和量化模型的下载请参考：[模型量化](../../quantize/README.md)

## 详细部署文档

在 RV1126 上只支持 C++ 的部署。
