# PaddleDetection

动态图版本的PaddleDetection, 此版本为试用版本，还在持续优化设计、性能、新增模型、文档等。


支持的模型:

- Faster-RCNN (FPN)
- Mask-RCNN (FPN)
- Cascade RCNN
- YOLOv3
- SSD
- FCOS
- SOLOv2
- PP-YOLO
- TTFNet

扩展特性：

- [√] **Synchronized Batch Norm**
- [√] **Modulated Deformable Convolution**
- [x] **Group Norm**
- [x] **Deformable PSRoI Pooling**
